Prototipo - TURISMO EDUCA

Instrucciones rápidas para empezar a desarrollar con Node en este proyecto.

Requisitos:

- Tener nvm instalado (ya lo configuramos).
- Terminal con zsh (recomendado).

Pasos iniciales:

1. Abrir el proyecto:
   cd '/Users/carinaheredia/TURISMO EDUCA'

2. Usar la versión indicada en .nvmrc:
   nvm use

3. Instalar dependencias (si hay):
   npm install

4. Ejecutar el prototipo en modo desarrollo:
   npm start

Notas:

- Si la terminal integrada de VS Code no carga nvm, asegúrate de que ~/.zshrc contiene las líneas que el instalador de nvm añadió. Cierra y abre la terminal/VS Code después de instalar nvm.
- .nvmrc contiene la versión LTS (24.11.0). Cambia si necesitas otra versión.
